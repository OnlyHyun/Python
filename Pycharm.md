# Pycharm Shortcuts 

## 1) 현재창실행 : ctrl + shift + f10

##    재실행     : shift + f10
   
##    선택 실행  : alt + shift + f10

## 2) 되돌리기   : ctrl + z

## 3) 복사       : ctrl + c

## 4) 붙여넣기   : ctrl + v

## 5) 폴더, 새파일 생성 : alt + insert

## 6) 편집창 닫기 : ctrl + f4
  
##    파이참 종료 :  alt + f4
   
## 7) 파일, 폴더이름 변경 : shift + f6 

## 8) 주석처리 : ctrl + / 

## 9) 소괄호 밖 탈출 : ctrl + shift + enter 

## 10) 블럭단위로 선택 : ctrl + w
